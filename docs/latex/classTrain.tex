\doxysubsection{Detailed Description}
Class representing a train transportation system. 

This class extends the \doxylink{classTransportationSystem}{Transportation\+System} and provides an implementation for moving citizens using train transportation. It includes information about the speed and cost associated with the train.

Moves a citizen using the train if they can afford the fare. 

This function checks the bank balance of the citizen. If the balance is greater than the train fare, the citizen will use the train for transportation. Otherwise, a message indicating insufficient funds is displayed.
