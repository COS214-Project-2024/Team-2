\doxysubsection{Detailed Description}
Abstract base class representing a citizen in the transportation system. 

This class serves as an interface for concrete implementations of citizens, providing the necessary methods to manage their attributes such as housing and employment status. \doxylink{classConcrete}{Concrete} subclasses must implement these virtual methods.
