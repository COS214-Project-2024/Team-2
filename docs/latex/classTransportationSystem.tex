\doxysubsection{Detailed Description}
Abstract base class for transportation systems. 

This class defines the interface for all transportation systems. Any class that derives from \doxylink{classTransportationSystem}{Transportation\+System} must implement the move method, which facilitates the movement of a citizen using a specific transportation mode.
