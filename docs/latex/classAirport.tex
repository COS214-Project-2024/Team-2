\doxysubsection{Detailed Description}
Represents an airport transportation system. 

The \doxylink{classAirport}{Airport} class inherits from \doxylink{classTransportationSystem}{Transportation\+System} and provides the functionality for citizens to use airport transportation if they have sufficient funds.
