\doxysubsection{Detailed Description}
A factory class for creating \doxylink{classConcreteCitizen}{Concrete\+Citizen} instances. 

This class implements the \doxylink{classCitizenFactory}{Citizen\+Factory} interface and is responsible for creating \doxylink{classConcreteCitizen}{Concrete\+Citizen} objects, providing a way to create citizens associated with specific buildings and government.
