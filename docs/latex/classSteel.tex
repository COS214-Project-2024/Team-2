\doxysubsection{Detailed Description}
Class representing the \doxylink{classSteel}{Steel} material resource. 

The \doxylink{classSteel}{Steel} class is derived from the \doxylink{classMaterials}{Materials} class and manages the quantity and state of steel resources. It implements methods for using and generating steel while handling its current state.
