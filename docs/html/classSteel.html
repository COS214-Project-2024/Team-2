Detailed Description

Class representing the Steel material resource.

The Steel class is derived from the Materials class and manages the quantity and state of steel resources. It implements methods for using and generating steel while handling its current state.

Generates a specified quantity of steel.

This method increases the quantity of steel and updates the state based on the new quantity.

Parameters
quantityThe amount of steel to generate.

This method increases the quantity of steel and updates the state based on the new quantity. If the quantity exceeds the maximum capacity, it's capped at that maximum, and the state is set to FullState. If it's positive but less than maximum, it's set to NormalState; if it is zero or less, it's set to EmptyState.

Uses a specified quantity of steel.

This method decreases the quantity of steel and updates the state based on the remaining quantity.

This method decreases the quantity of steel and updates the state based on the remaining quantity. If the quantity becomes zero or negative, the state is set to EmptyState. If it's less than the maximum capacity, it's set to NormalState; otherwise, it's set to FullState.
