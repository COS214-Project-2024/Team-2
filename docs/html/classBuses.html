void Buses::move ( Bank *  bank)
virtual

Moves a citizen using the bus transportation system.

Executes the bus movement logic.

This method checks if the citizen has enough funds in the bank to use the bus. If they do, it prints a success message and deducts the price of the bus fare from their bank balance. Otherwise, it informs them that they do not have enough money.

Parameters
bankA pointer to the Bank object representing the citizen's bank.

This method checks the user's bank balance and determines whether they can afford to use the bus. It updates the balance accordingly and provides feedback to the user about the transaction.
