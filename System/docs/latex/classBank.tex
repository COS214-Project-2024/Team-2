\doxysubsection{Detailed Description}
Represents a bank account for managing a citizen\textquotesingle{}s funds. 

The \doxylink{classBank}{Bank} class provides functionality to get the current balance, increment or decrement the balance, and automatically adjust the budget based on the balance.
