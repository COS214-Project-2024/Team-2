\doxysubsection{Detailed Description}
Represents a concrete material in the resource management system. 

The \doxylink{classConcrete}{Concrete} class manages the quantity of concrete resources, and tracks its state using various {\ttfamily \doxylink{classStateOfResources}{State\+Of\+Resources}}. It supports resource usage and generation, as well as state transitions.
