\doxysubsection{Detailed Description}
Represents a \doxylink{classWood}{Wood} resource in the system. 

The \doxylink{classWood}{Wood} class is derived from the \doxylink{classMaterials}{Materials} class and manages the quantity of wood resources and their states (Full, Empty, Normal). It allows for using and generating wood resources while ensuring the state transitions based on the current quantity.

Generates a specified quantity of wood. 

This method increases the quantity of wood and updates the state based on the new quantity. If the quantity exceeds the maximum capacity, it is capped at that maximum, and the state is set to \doxylink{classFullState}{Full\+State}. If it is positive but less than maximum, it is set to \doxylink{classNormalState}{Normal\+State}; if it is zero or less, it is set to \doxylink{classEmptyState}{Empty\+State}.
